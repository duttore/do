## Analisi SEO - Struttura Cartella

Questa cartella contiene:

- `keywords/keywords.csv`: dataset delle query con metriche (click, impression, CTR, posizione).
- `keywords/analysis.md`: insight operativi e priorità sulle keyword.
- `seo_audit.md`: audit SEO delle pagine del sito con checklist e azioni.

Come usare:
- Aggiorna `keywords.csv` quando arrivano nuovi dati.
- Tieni traccia delle modifiche e degli impatti nelle sezioni “Azioni Prioritarie”.
